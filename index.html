What is this?

This is a small tool that lets you save ChatGPT conversations to your computer with a single click, using a JavaScript bookmarklet.

Originally introduced in a DEV.to article, it is now maintained on GitHub to ensure continued compatibility with changes in OpenAI's UI.

Why does this exist?

Since ChatGPT’s interface can change at any time, this tool uses a CDN-hosted script from GitHub. This keeps it working even when OpenAI updates their web UI.

How do I use it?

To run the code you need to create a new bookmark and copy the code from the JavaScript file.

If you don't know how, you can drag & drop the Save ChatGPT link to your bookmarks.
